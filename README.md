# Gorila Carteira API

![alt text](https://gorila.com.br/wp-content/uploads/LogoGorila.svg)

## Description

API REST com autenticação via JWT(Json Web Token) para o projeto teste carteira gorila

## Instalação

```bash
$ npm install
```

## Iniciando a api

```bash
# dev mode
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
